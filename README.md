# NNIT-ai-freelance-backend

Repository bootstrap examples for possible stacks. This repo contains small example bootstraps (see branches like `feat/bootstrap-nodejs`) demonstrating minimal services, tests, and CI workflows to help maintainers evaluate stacks.

## Getting started (Node.js example)

- **Node:** recommended Node 18+.
- **Install dependencies:** `npm install`
- **Run locally:** `npm start` (defaults to port 3000 or uses `$PORT`)
- **Run tests:** `npm test` (uses `jest --runInBand`)

**Contributions:**
- When proposing a bootstrap, keep PRs small (one service + tests + CI). See `CONTRIBUTING.md` for details.
